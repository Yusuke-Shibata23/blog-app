# 『三体』を読む - 中国SFの金字塔

## はじめに

劉慈欣の『三体』三部作は、人類の運命を賭けた壮大なSF作品です。この記事では、作品の魅力と深い哲学的考察について解説します。

*『三体』の表紙*

### ストーリー
- 文化大革命時代から始まる物語
- 地球外文明との接触
- 人類の存続を賭けた戦い

### 主要人物
- 葉文潔
- 汪淼
- 史強
- 程心

### 第一部：三体
- 地球外文明の発見

- 宇宙の終焉
- 人類の未来
